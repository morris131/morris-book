## 安装JDK并配置Java环境变量
Maven 是一个基于 Java 的工具，所以要做的第一件事情就是安装 JDK。

cmd下验证是否配置成功：
```bash
>java -version
java version "1.8.0_101"
Java(TM) SE Runtime Environment (build 1.8.0_101-b13)
Java HotSpot(TM) 64-Bit Server VM (build 25.101-b13, mixed mode

>javac -version
javac 1.8.0_101
```

## 下载Maven并配置Maven环境变量
下载地址：http://maven.apache.org/

下载完后将压缩包解压到一个目录，如D:\development。

设置环境变量。
```bash
变量名：M2_HOME
变量值：D:\development\maven\apache-maven-3.3.9
```
在环境变量path中添加：
```bash
%M2_HOME%\bin
```
cmd下验证是否配置成功:
